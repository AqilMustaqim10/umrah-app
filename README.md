# 🕋 Umrah Companion

A modern full-stack web application to help Muslims prepare for and perform Umrah.

## Features

- 🔐 Secure Authentication (Register, Login, Forgot Password)
- ✅ Umrah Steps Checklist
- 🎒 Packing Checklist with Categories
- 👤 User Profile Management
- 📱 PWA — Installable on Mobile

## Tech Stack

### Frontend

- React + Vite
- TailwindCSS
- Framer Motion
- React Hook Form
- React Router

### Backend

- Node.js + Express
- MongoDB Atlas + Mongoose
- JWT Authentication
- bcrypt
- Nodemailer

## Deployment

- Frontend → Vercel
- Backend → Render
- Database → MongoDB Atlas
